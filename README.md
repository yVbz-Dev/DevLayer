
# DevLayer

DevLayer is a open-source social media so developers can, interact, share ideas, and much more!

# Status
In development.

## Tech Stack

**Client:** React

**Server:** Node, Express

**Database:** Supabase

**Auth:** JWT, Bcrypt

## FAQ

#### Is this a serious project?

No. This is supposed to be a learning project.

#### Why use node.js?

Its easy to setup.

#### Why use react?

Because it makes building websites easier.


#### Why use supabase?

Its very easy to setup supabase.
